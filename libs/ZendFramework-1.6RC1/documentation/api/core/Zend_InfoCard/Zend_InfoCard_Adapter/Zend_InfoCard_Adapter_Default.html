Zend_InfoCard_Adapter_Default

Description
Description | Methods (details)

Implements interfaces:

  • Zend_InfoCard_Adapter_Interface

The default InfoCard component Adapter which serves as a pass-thru placeholder

for developers. Initially developed to provide a callback mechanism to store and retrieve assertions as part of the validation process it can be used anytime callback facilities are necessary
