Zend_Service_Technorati_TagsResult

Description
Description | Vars (details) | Methods (details)

Represents a single Technorati TopTags or BlogPostTags query result object.

It is never returned as a standalone object, but it always belongs to a valid Zend_Service_Technorati_TagsResultSet object.
